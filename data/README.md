# Data layout

Downloaded images and generated artifacts are intentionally not versioned.

```text
data/
  raw/PIID/original_images/{1,2,3,4}/
  raw/Kaggle/original_images/{Stage_I,Stage_II,Stage_III,Stage_IV}/
  processed/analytic_data/{PIID,Kaggle,HUMC}/{1,2,3,4}/
  private/HUMC/labels/humc_labels.xlsx
  splits/piid_main/
  splits/humc_patient_level/
  results/
```

PIID and Kaggle analytic images are generated by `code/data_curation/prepare_public_datasets.py`. The script corrects orientation, center-crops the largest square around the midpoint of the long axis, and resizes to 224 x 224.

HUMC images, labels, patient identifiers, and image-level splits remain local and are covered by `.gitignore`. Only privacy-safe aggregate metadata and fold-wise normalization values are released.
