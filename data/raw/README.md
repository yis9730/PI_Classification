# Raw public datasets

Download each dataset from its provider; image files are not redistributed by this repository.

PIID source: <https://drive.google.com/drive/u/0/folders/12JouktrzXIo6ywpSe2OYWRYNNIxlEKvK>

```text
data/raw/PIID/original_images/
  1/
  2/
  3/
  4/
```

The curation manifest expects 1,091 four-stage PIID images before exclusion.

Kaggle source: <https://www.kaggle.com/datasets/sinemgokoz/pressure-ulcers-stages>

```text
data/raw/Kaggle/original_images/
  Stage_I/
  Stage_II/
  Stage_III/
  Stage_IV/
```

`Invalid`, `SDTI`, and `Unstageable` are outside the four-class study.
